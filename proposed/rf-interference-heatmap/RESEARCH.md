# Satellite RF Interference Heatmap — Research Notes

**Generated:** 2026-06-05
**Category:** atmosphere
**Manager:** RfiHeatmapManager.js

## Strategic Value
Space-based RF interference detection is effectively open-source ELINT derived from NASA science data. SMAP's L-band radiometer reveals undeclared radar emitters, GPS/GNSS jammers, and active electronic warfare zones worldwide — the Ukraine conflict produced massive observable RFI signatures from drone countermeasures visible from orbit. Correlating persistent RFI hotspots with military installations, conflict boundaries, shipping lanes, and the existing gpsJammingManager ground-truth data creates a multi-source electronic order of battle that no single sensor provides alone. This gives analysts a dimension of intelligence — the electromagnetic environment — that is otherwise invisible on a map.

## Data Source
NASA SMAP L1B/L3 RFI data via NSIDC DAAC: https://nsidc.org/data/spl1btb/versions/3 (L1B brightness temperatures with RFI flags) and pre-processed RFI maps at https://smap.jpl.nasa.gov/rfi/ providing 0.25°×0.25° gridded RFI persistence as percentage over 4-week windows. Programmatic access via NASA CMR API: https://cmr.earthdata.nasa.gov/search/granules.json?collection_concept_id=C2531308461-NSIDC_ECS&temporal=2024-01-01T00:00:00Z,2024-12-31T23:59:59Z — returns granule download links. NASA Earthdata bearer token auth (free registration at https://urs.earthdata.nasa.gov/). For near-real-time complementary data: Spire RF tasking API at https://documentation.spire.com/ provides targeted RF collections in IQ/JSON/CSV format (commercial, requires contract).

## Visual Concept
A translucent heatmap overlay draped across the globe's surface showing RF interference intensity. Clean spectrum appears as fully transparent; low RFI as cool blue shimmer; moderate interference as amber glow; intense persistent RFI as angry red-orange hotspots that pulse slowly. The layer sits at Y≈0.08, just above water and terrain surfaces, with depth-based opacity falloff so it fades when the camera is far away and intensifies on zoom. Known military installations and conflict zones light up as bright emission clusters. A temporal scrubber allows analysts to animate RFI persistence week-by-week, revealing the electromagnetic signature of conflict escalation — Ukraine, Middle East, South China Sea hotspots visibly intensify over time. When combined with gpsJammingManager's ground-reported jamming circles, analysts see both the cause (RFI emitters from space) and effect (GPS disruption on the ground) simultaneously.

## Implementation Approach
Pre-process SMAP 0.25° RFI persistence grids into a 1440×720 single-channel Float32 DataTexture (one value per grid cell: RFI persistence percentage 0-100). Create a PlaneGeometry(worldWidth, worldHeight, 1, 1) positioned at Y=0.08 with custom ShaderMaterial. Vertex shader is passthrough; fragment shader samples the RFI DataTexture at UV derived from world XZ position via sceneToLonLat(), applies a configurable color ramp uniform (vec3 array: transparent→blue→amber→red), and outputs with alpha proportional to RFI intensity multiplied by a distance-based fade uniform updated from camera position. Add slow pulsing animation for cells above a HIGH_RFI_THRESHOLD via sin(time) modulating emission. For temporal animation, load multiple monthly DataTextures into a texture array and lerp between adjacent months based on a timeline uniform driven by UI scrubber. Dispatch 'vg1:rfi-heatmap:month-changed' when temporal index updates so gpsJammingManager can synchronize its display period. Mask land vs ocean using the existing terrain/water boundary data to avoid rendering over terrain where terrainBuilder already provides surface detail — or render over both with additive blending (THREE.AdditiveBlending) so it composites as a glow. Register with layerManager under 'atmosphere' group. Config constants: RFI_HEATMAP_OPACITY_MAX (0.6), RFI_HEATMAP_HIGH_THRESHOLD (40.0 — persistence %), RFI_HEATMAP_PULSE_SPEED (0.8), RFI_HEATMAP_COLOR_RAMP ([0x000000, 0x1a3a5c, 0xd48a2e, 0xff2200]), RFI_HEATMAP_ALTITUDE (0.08).

## Research Notes
SMAP (Soil Moisture Active Passive) launched 2015, operates at 1.4 GHz L-band which is a protected radio astronomy band — any signal detected here is by definition interference, making RFI extraction unambiguous. JPL publishes pre-computed RFI maps at https://smap.jpl.nasa.gov/rfi/ showing global persistence grids on 0.25° resolution, updated regularly. The Ukraine conflict produced dramatic observable RFI increases visible in SMAP data as both sides deployed drone jammers and EW systems operating near L-band. SMAP RFI is measured as the delta in Kelvin between pre- and post-RFI-filtered antenna temperatures; persistence is the percentage of measurements flagged over a 4-week window. Data is distributed as HDF5 files via NSIDC — h5wasm (https://github.com/usnistgov/h5wasm) enables browser-side HDF5 parsing, or pre-process server-side to PNG/binary. The 1440×720 grid at Float32 is only ~4MB per monthly snapshot, making client-side multi-month animation feasible with <50MB total. This complements gpsJammingManager (which shows receiver-reported GNSS disruption from gpsjam.org) by providing the emitter-side view from space — the two layers together create a complete EW picture. For higher-fidelity commercial RF data, Spire's constellation provides targeted RF collection but requires commercial agreement. CelesTrak also tracks some known jamming-capable satellite payloads that could be cross-referenced.
